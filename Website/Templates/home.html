My Cryptography Game


In school, I have a class called ASE and we were tasked with making a logic game. Logic game is a broad category so we could do basically whatever we wanted to as long as it provoked cognitive thinking. I decided to make a cryptogram because I was researching the Zodiac killer for an English project and he used cryptograms to taught the police. So I said hey why not. I wanted to make a website so that I could control automate everything and that's what I did. I learned HTML, javascript, and CSS in the process and this is what I ended up with. Enjoy!
